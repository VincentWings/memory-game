# Memory Game
The Memory Game is a card matching game project made for the completion of the Udacity's Front End Developer Nanodegree.

## Instructions
```
Open https://vincentwings.github.io/memory-game/ 
or Download / clone the repository :
https://github.com/VincentWings/memory-game.git and open index.html to start playing.
```

## Rules

1. Match all the cards!
2. You rating may vary depending on the number of moves you make.
3. Good luck
